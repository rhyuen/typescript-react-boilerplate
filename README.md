# LESSONS LEARNED

*November 6, 2019*
1. The Max Age field in Cookies for Javascript is in Milliseconds not Seconds, despite what the docs say.

> 
```javascript 
    {
        maxAge: 3600 //this is 3.6 seconds
    } 
```
